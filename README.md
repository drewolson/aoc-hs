# Advent of Code

[![test](https://github.com/drewolson/aoc/actions/workflows/test.yml/badge.svg)](https://github.com/drewolson/aoc/actions/workflows/test.yml)

## Build

```
stack build
```

## Run Solution

```
stack run -- --help
stack run -- --day 1 --part 1
stack run -- -d 1 -p 1
```

## Test

```
stack test
```

## Format

```
make format-check
make format
```
